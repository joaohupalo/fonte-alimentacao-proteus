# fonte-alimentacao-proteus
Repositório do projeto de fonte linear 12V no Proteus. Inclui esquemático, layout de PCB e visualização 3D. O circuito utiliza retificação em ponte (BR1), filtragem (C3 1000µF), regulação (CI 7812) e sinalização por LED para converter entrada AC em saída DC estável. Ideal para fins acadêmicos.
