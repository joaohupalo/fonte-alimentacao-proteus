Fonte de Alimentação Linear Regulada 12V

Este projeto consiste no desenvolvimento de uma fonte de alimentação linear estável, projetada no software Proteus. O sistema realiza a conversão de tensão alternada (AC) para uma saída contínua (DC) fixa de 12V.
Descrição do Circuito

O funcionamento segue as etapas fundamentais de uma fonte linear:

    Retificação: A ponte BR1 converte a entrada AC do conector J2 em corrente contínua pulsante.

    Filtragem: O capacitor eletrolítico C3 (1000µF) suaviza a ondulação da tensão.

    Regulação: O CI 7812 (U1) estabiliza a saída em 12V nominais.

    Sinalização: O LED D1, limitado pelo resistor R1 (100Ω), indica a presença de tensão na saída.

    Saída: Conector J1 para alimentação de cargas externas.
